# data-analysis

Claude Code скил для систематического анализа CSV/Excel: профилирование → аномалии → описательная статистика → паттерны → executive summary с обязательными PNG-графиками.

## Установка

```text
/plugin marketplace add rukru/claude-skills
/plugin install data-analysis@rukru-claude-skills
```

Полный гайд — в [README репо](../../README.md).

## Что делает

При запросе вроде «проанализируй этот csv» скил последовательно прогоняет 5 шагов:

| Шаг | Что делает |
|---|---|
| 1 | Профилирование: размер, типы, пропуски, уникальные значения |
| 2 | Поиск аномалий ДО гипотез — выбросы, временные скачки, противоречия, дубликаты |
| 3 | Описательная статистика — распределения, группировки |
| 4 | Поиск паттернов — когорты, корреляции, Simpson's Paradox |
| 5 | Executive Summary — 3 вывода + 1 инсайт + рекомендация + вопрос |

## Что на выходе

Самодостаточная папка анализа:

```
{report_dir}/
├── report.md          # markdown с YAML frontmatter
├── R1_*.png           # графики (PNG, dpi=150)
├── R2_*.png
├── analysis.py        # автономный скрипт (pandas+matplotlib+seaborn)
└── processed.csv      # обработанные данные (опционально)
```

Графики **обязательны** — для каждой ключевой таблицы в отчёте создаётся визуализация.

## Зависимости

Для реальной генерации графиков скилу нужен Python:

```bash
pip install pandas matplotlib seaborn
```
